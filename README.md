# Naver Moive Review Crawling

### Introduction

네이버 영화 사이트 서비스트 종료 이후 네이버에서 제공하는 리뷰를 크롤링 할 수 있는 코드

### Requirements

selenium == 4.10.0

pymysql >= 1.0.2, <= 1.1.0
